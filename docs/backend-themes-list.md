## How to Use

1. Navigate to **Settings > Backend Appearance**
2. Click on any theme card to select it
3. The theme applies immediately to all admin pages

## Theme Characteristics

Each theme includes:
- **Colors**: Background, surface, primary, text, borders, status colors
- **Typography**: Font family, sizes, line heights
- **Radius**: Border radius values (small, medium, large)
- **Shadows**: Card, elevated, and subtle shadow definitions

All themes are fully customizable and can be edited through the theme management API.
